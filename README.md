# Next.js with TypeScript 


Live Demo http://crema-next-ts.herokuapp.com/

Install it and run:

```sh
npm install
npm run dev
```

## The idea behind the example

[Next.js](https://github.com/zeit/next.js) is a framework for server-rendered React apps.
